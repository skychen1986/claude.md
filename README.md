# Claude Code 配置模板（脱敏版）

这是我的 Claude Code 和 Obsidian 知识库配置文件，脱敏后公开分享。

## 包含什么

- **Claude 全局规则**：我是怎么"调教" Claude 的——先反问再执行、独立判断据理力争、代码交付标准
- **Obsidian ABCD 知识库**：四层自生长循环（原始素材 → 提炼知识 → 技能库 → 产出回流），让你的笔记不是"收藏夹"而是"第二大脑"

## 怎么用

1. 下载 `CLAUDE.md`
2. 把"关于我"那部分换成你自己的信息
3. 全局规则放 `~/.claude/CLAUDE.md`
4. ABCD 知识库部分放你的 Obsidian vault 根目录

看我那篇 Obsidian 口播稿——讲了怎么让 Claude 帮你维护一个会自动生长的知识库。
